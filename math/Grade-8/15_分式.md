# 概念
一般地，如果A, B表示两个整式，并且B中含有字母, 那么式子$\frac {B}{A}$叫做分式

在分式$\frac {B}{A}$中，A叫做分子，B叫做分母

> 1. 分式有意义，就是要分母不为0
> 2. 分式的值为0必须满足两个条件： ①分子的值为0；②分母的值不等于0
> 3. 分式的学习类比分数，从除法的角度考虑。

# 基本性质
分式的分子与分母同乘（或除以）一个不等于0的整式，分式的值不变

公式表示

$\frac {A}{B}$ = $\frac {A*C}{B*C}$，$\frac {A}{B}$ = $\frac {A÷C}{B÷C}$，其中C是不等于零的整式

## 应用分式的基本性质需要注意
1. 分子、分母应同时做乘、除法中的同一种运算；
2. 所乘（或除以）的必须是同一个整式； 
3. 所乘（或除以）的整式应该不等于0.

# 变号法则
分式本身及其分子、分母这三处的正负号中，同时改变两处，分式的值不改变
> 分式的学习类比分数的学习，可以借助图形来提升对于代数的理解。

# 分式的约分
根据分式的基本性质，把一个分式的分子和分母的公因式约去，叫做分式的约分． 

像$\frac {3}{4x}$这类分子和分母没有公因式的分式，叫做最简分式. 

> 关键是确定公因式。确定公因式，系数取最大公约数，字母取相同字母的最低次幂

# 分式的通分
根据分式的基本性质，把几个异分母的分式化成与原来的分式值相等的同分母的分式叫做分式的通分.

## 最简公分母的确定
1. 各分母系数的最小公倍数 
2. 相同字母因式的最高次幂的积

# 分式的乘法
## 乘法法则
分式乘分式，用分子的积作为积的分子，分母的积作为积的分母. 

可以用式子表示为：$\frac {a}{b}$·$\frac {c}{d}$ = $\frac {a·c}{b·d}$
>注意事项:
>1. 当分式的分子、分母是多项式时，要先分解因式
>2. 运算的结果一定化为最简分式

# 分式的除法
## 除法法则
分式除以分式，把除式的分子、分母颠倒位置后，与被除式相乘

可以用式子表示为：$\frac {a}{b}$÷$\frac {c}{d}$ = $\frac {a}{b}$·$\frac {d}{c}$ = $\frac {a·d}{b·c}$

## 注意事项
分式的乘除混合运算，统一为乘法运算，若运算中含有括号先算括号里面的，再按照从左到右的顺序进行计算

# 分式的乘方
## 乘方法则
分式乘方要把分子、分母分别乘方

可以用式子表示为：$(\frac {a}{b})^n$ = $\frac {a^n}{b^n}$

# 分式的加法
## 同分母的分式
同分母的分式相加，分母不变，把分子相加

## 异分母的分式
异分母的分式相加，先通分，变为同分母的分数，再相加

## 最简公分母
取各分母的所有因式的最高次幂的积作为公分母

### 方法
1. 系数：把各分式分母系数的最小公倍数作为最简公分母的系数；
2. 字母因式：所有出现过的字母因式（或因式分解后得到的）都要取到；
3. 指数：相同因式取指数最高的

# 分式的减法
## 减法法则
同分母的分式相减，分母不变，把分子相减

异分母的分式相减，先通分，变为同分母的分式，再相减

# 整数指数幂
> 复习参考 14_整式的乘法与因式分解.md
## 负整数指数幂的意义
在数学上，我们规定：

当n为正整数时，${a^{-n}}$ = $\frac {1}{a^n}$ (a $\ne$ 0)

## 计算性质

1. $a^m$·$a^n$=$a^{m+n}$（m,n为整数）
2. $(a^m)^n$=$a^{mn}$（m,n为整数）
3. $(ab)^n$=$a^{n}$$b^{n}$（n为整数）
4. $a^m$÷$a^n$=$a^{m-n}$（a不等于0，m,n为整数）
5. $(\frac {a}{b})^n$ = $\frac {a^n}{b^n}$（n为整数）
6. $a^0$=1（a不等于0）

## 科学计数法的推广
绝对值小于1的数可以用科学记数法表示为ax$10^{-n}$ 的形式，其中1≤|a|<10，n是正整数

0.002，即 2x$10^{-3}$

# 分式方程
## 定义
分母中含有未知数的方程，叫做分式方程

## 解分式方程
### 基本思路
将分式方程化为整式方程.

### 具体做法
去分母，方程左右两边同乘最简公分母.

### 步骤
1. 去分母（两边同乘最简公分母）
2. 解整式方程
3. 检验（代入最简公分母，为0，不是分式方程的解，否则是分式方程的解）
4. 写出原分式方程的解

### 注意事项
1. 解较复杂分式方程时，先变形整理，能约分的先约分，可方便确定最简公分母；
2. 注意每一步变形都要有依据，去分母时，不要漏乘不含分母的项；
3. 互为相反数的多项式，可通过对其中一个提取负号实现统一
4. 解分式方程一定要检验
