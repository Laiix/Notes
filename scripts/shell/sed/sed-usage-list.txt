Command:
sed -n '/package/{h;d};/import/!{$!N;P;/\nimport/{s/.*\n//;H;x;p;b};D};{p}' file

Fucntion:
调整如下内容：
package com.eussi;
//author
//date

import com.eussi.Sed;
import com.eussi.Test;
import com.eussi.A;

public class Sed {

}
将package一行移动插入至第一个import行前面

Notice:
1、经测试，如果直接改写文件，-i命令需放在-n命令后
2、另外，如果不存在import行，package行会丢失
